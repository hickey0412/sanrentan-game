# 🐴 サンレンタン (Sanrentan Web)

人気ボードゲーム「サンレンタン」をWebブラウザで遊べるようにしたパーティーゲームアプリです。
PCを親機（モニター）とし、参加者が各自のスマートフォンから回答を送信してリアルタイムに遊ぶことができます。

## 🎮 ゲームの概要
親プレイヤーがお題に対する「ベスト3」を決め、子プレイヤーたちがその順位を予想します。
正解数に応じて「サンレンタン（全正解）」「サンレンプク」などの点数が入り、合計得点を競います。

## ✨ 特徴
- **リアルタイム連携**: 親機の画面と参加者のスマホが同期して進行します。
- **QRコード参加**: 参加者はQRコードを読み取るだけでルームに入室可能。
- **自動採点**: 回答が揃うと自動で答え合わせと点数計算が行われます。
- **履歴機能**: ゲームの結果とスコアの推移が記録されます。

## 🛠️ 動作環境
- Python 3.x
- Flask
